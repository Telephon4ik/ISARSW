Описание

ЖД Диспетчер - это десктопное приложение для управления расписанием поездов на железнодорожных станциях. Приложение предоставляет полный набор инструментов для диспетчеров, включая создание, редактирование, отслеживание статусов и предотвращение конфликтов расписания.

Основные возможности
* Управление поездами: Создание, редактирование и удаление записей о поездах
* Контроль платформ: Автоматическое предотвращение конфликтов использования платформ
* Автоматические проверки: Система автоматически проверяет прибытие и отправление поездов
* Мощный поиск и фильтрация: Быстрый поиск поездов по различным критериям
* История изменений: Полный аудит всех изменений по каждому поезду
* Интуитивный интерфейс: Современный интерфейс с поддержкой тем

Установка

Требования

* Java 17 или выше
* Операционная система: Windows 10/11, macOS, Linux
* Память: Минимум 2 ГБ ОЗУ
* Дисковое пространство: 500 МБ

Шаги установки

* Установите Java 17+ если не установлена
* Скачайте приложение:
* Скачайте последнюю версию ISARSW.jar из раздела Releases
* Запустите приложение

Первоначальная настройка:
* При первом запуске автоматически создастся база данных
* Начните с создания тестовых поездов
* 
Использование

Быстрый старт

* Запустите приложение
* Создайте первый поезд:
* Нажмите "Новый поезд"
* Заполните все обязательные поля
* Нажмите "Создать"

Отслеживайте статусы:
* Статусы автоматически меняются по расписанию
* При необходимости измените статус вручную через окно проверки

Используйте фильтры:
* Нажмите "Фильтры" для настройки отображения
* Используйте поле поиска для быстрого поиска

Разработка

Сборка из исходников

Клонируйте репозиторий:

cd ISARSW***

Зависимости

* JavaFX 17+ - для графического интерфейса
* SQLite JDBC - для работы с базой данных
* ControlsFX - дополнительные компоненты UI

Структура кода

* MVC архитектура: Четкое разделение на Model, View, Controller
* Сервисный слой: Вся бизнес-логика в сервисах
* DAO паттерн: Изолированный доступ к данным
* Событийно-ориентированная архитектура: Асинхронные проверки и уведомления
